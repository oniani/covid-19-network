## Quickstart

```sh
# Install dependencies
python3 -m pip install -r requirements.txt

# Generate data
python3 generate_graph_data.py
python3 generate_edges_features_data.py

# Generate graph
python3 generate_graph.py

# Link prediction
python3 predict_links.py
```
